# GoodGain
Uma plataforma web de simulação de apostas esportivas e análise de dados, com o objetivo de proporcionar aos usuários conhecimento e ferramentas necessárias para melhorar suas chances de sucesso em apostas esportivas reais. Projeto de TCC desenvolvido durante o 7º e 8º período do curso de Bacharelado em Sistemas de Informação, da Pontifícia Universidade Católica do Paraná.
